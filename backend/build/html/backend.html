backend.app module¶

Flask Application for User Management

This module provides a Flask application that supports user registration, login, and retrieval of user details. It uses Flask-SQLAlchemy for database operations, Flask CLI for command line commands, and includes secure password handling.

backend.app.register()[source]¶

Register a new user.

Endpoint: /register Method: POST

Request JSON Body:
{

“email”: str, # User’s email address. “username”: str, # User’s desired username. “password”: str # User’s plain text password.

}

Returns:

  • 201: {“message”: “User created successfully”} on successful registration.

  • 400: {“error”: “Missing required fields”} if any field is absent.

  • 400: {“error”: “User with given email or username already exists”} if duplicate user.

Return type:

JSON response with

Note

This endpoint adheres to the Information Hiding Principle by exposing only the necessary contract (input fields and expected responses) while encapsulating internal logic such as password hashing.
